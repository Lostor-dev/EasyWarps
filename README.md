# EasyWarps: Удобный и Легкий Менеджер Варпов

**EasyWarps** — это простой и интуитивно понятный плагин для сервера Spigot/Paper, предназначенный для управления всеми варпами и спаунами через современный графический интерфейс (GUI). Забудьте о длинных командах — вся навигация теперь в несколько кликов!

## Основные Возможности

`Все можно настроить через config.yml`

**Современное GUI:** Доступ ко всем варпам и спаунам через удобное, полностью настраиваемое меню.

**Гибкая Фильтрация:** Быстрое переключение между категориями варпов прямо в меню:

**Server** (Серверные)

**Public** (Публичные, созданные игроками)

**My** (Ваши личные варпы)

**Spawns** (Все доступные точки спауна)

**Множество Спаунов:** Устанавливайте и используйте несколько точек спауна (_/setspawn 1_, _/setspawn 2_) с возможностью случайной телепортации.

**Безопасность:** Встроенная проверка локации перед телепортацией для предотвращения застревания игроков в блоках.

**Улучшение Команд:** Поддержка Автодополнения (Tab Completer) для всех команд, делая их использование максимально быстрым и удобным.

## Команды и Права

`Все права можно выставить через plugin.yml`
`Сурсы можно скачать на гитхаб`

`/warps [фильтр]`	Открывает GUI.	**easywarps.warps**

`/warp <название>`	Телепортирует к варпу.	**easywarps.warp**

`/setwarp <название>`	Установить новый варп. **easywarps.setwarp**

`/delwarp <название>`	Удалить варп.	**easywarps.delwarp**

`/spawn [номер]`	Телепортирует на спаун.	**easywarps.spawn**

`/setspawn [номер]` Устанавливает спавн с номером (необязательно). **easywarps.setspawn**

`/delspawn [номер]` Удаляет спавн с номером (необязательно). **easywarps.delspawn**
